groupentity

Table
Database: projectary

Esta tabela é responsável por agrupar vários entrevenientes alunos ou professores associados a um projecto.

NameData TypeComment
id id : varchar (255) : Not NULL Chave primária desta tabela.
entity entity : varchar (255) * : Not NULL Chave forasteira "groupentity_entity_fk" que relaciona um grupo a uma entidade.
function function : varchar (255) * : Not NULL Chave forasteira "group_function_fk" que relaciona um determinado membro do grupo a uma função desempenhada.
grade grade : decimal (10,0) : NULL Nota decimal que está atribuida a um determinado grupo de pessoas envolvidas num projecto.
createdin createdin : timestamp : Not NULL Registo temporal que indica quado a tabela foi criada.
createdby createdby : varchar (255) * : Not NULL Chave forasteira "groupentity_createdby_fk" que indica que entidade.
approvedby approvedby : varchar (255) * : NULL Chave forasteira "groupentity_approval_fk" que indica quem aprovou este grupo. (Um teacher com desempenhado uma function de admin).
approvedin approvedin : timestamp : NULL Registo temporal que indica quando foi aprovada a tabela.
